Subtitles to text

This is a tool that can convert subtitles (video captions) to readable text. It can convert one or more subtitle files, or even an archive containing subtitles in a folder structure.

When converting multiple files, their content will be put into chapters, based on the file structure, in alphabetical order.

The tool supports either SRT files, or ZIP files containing SRT files.

Convert

If you own a Kindle, you can select the Kindle friendly format, and send the result as an attachment to your Kindle e-mail address - it will be automatically converted into an e-book you can read on your device. More details here.
